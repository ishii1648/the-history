## Implementation Notes

<!-- SECTION:NOTES:BEGIN -->
調査結果（Web 調査 subagent、2026-07-23。詳細な根拠 URL 付き評価は以下）:

候補評価の要約:
- ETH Roller（既採用, CC BY-NC-SA 4.0）: 原資料（Schindling & Walter 1989）が 1500–1650 専用の学術地図集のため、900〜1492 への拡張版は存在しない。
- Euratlas Periodis 'Provinces' レイヤー: 唯一「年代（900〜1400 含む 100 年刻み）× 領邦粒度（公国・伯領）」を満たすが、商用ライセンス（150〜600 ユーロ、内部使用限定・再配布/改変不可、Stanford EarthWorks 版も学外配布不可）で、本アプリの取得→変換→静的 GeoJSON 配信パイプラインは再配布に該当し組み込み不可。
- HGIS Germany: 1648 年以降のみ。HistoGIS（帝国クライス）: 1512 年以降かつ粒度が粗い。DARMC: HRE 領邦粒度の系統的シリーズなし。GeaCron: データ配布経路なし。Wikimedia SVG: 地理座標なしで手動ジオリファレンス必須（既存オープンデータの要件を満たさない）。
- OpenHistoricalMap 'Holy Roman Empire' プロジェクト（CC0 志向、Overpass API で empire=hre 抽出可）: 方向性は最有望だが Wiki 上 'In progress, recruiting helpers' であり 900〜1492 の領邦境界は未整備。現時点では成熟度不足、将来の再評価候補。

結論: 採用可能なデータソースは現状存在しない。900〜1492 年は単一ポリゴン表示の現状維持が妥当（データのでっち上げはしない）。コード変更なし。参照 URL は調査 subagent の成果として本ノート末尾のタスク説明・PR 説明に記載。

<!-- SECTION:FINAL_SUMMARY:BEGIN -->
